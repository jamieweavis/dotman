## Installation

Dotman is installed globally via the command line from the npm registry with either [yarn](https://github.com/yarnpkg/yarn) or [npm](https://github.com/npm/npm).

```sh
# Via yarn
$ yarn global add dotman

# Via npm
$ npm install --global dotman
```
